La page web est obtenue par interprétation d'un code HTML envoyé par le serveur en réponse à la requête qui lui a été envoyée.C'est le navigateur qui interprète ce code.

Il existe deux sortes de balises :

  1. les balises les plus courantes encadrent du code qui se termine par une balise dite fermante. Exemple:<balise> contenu HTML </balise>
    2. &
  2. les balises dites orphelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple:<br> <hr> <img> <imput> etc...
